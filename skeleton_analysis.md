# 🦴 视频骨架结构规律分析报告

> 基于素材库 120 条视频骨架数据的深度拆解

---

## 一、骨架格式的统一规律

所有骨架都遵循同一套**事件链语法**：

```
事件描述(时长s) → 事件描述(时长s) → ... → 结尾事件(时长s)
```

- **箭头 `→`** 是唯一分隔符，代表"然后发生"
- **括号 `(Ns)`** 标注每个节拍的秒数
- 每条骨架通常包含 **5~12 个节拍**，总时长集中在 **15s~55s**
- 语言风格是**"动词驱动的画面描述"**，没有对白只有动作

---

## 二、七大叙事公式

通过对 120 条骨架的归纳，可以提炼出 **7 种反复出现的核心结构公式**：

---

### 公式①：递进循环式（出现约 30%）

> **A→A'→A''→…→爆发/崩坏**

同一个动作/模式不断重复，每轮升级一个维度，直到最后一轮崩坏或反转。

| 子类型 | 代表视频 | 结构 |
|--------|----------|------|
| **策略全败循环** | #18 大小道具对决、#40 不公平竞赛、#74 四轮竞技、#96 三轮挑战、#100、#112 | 第一轮A选错败→第二轮A再选错败→…→最终颁奖A全败 |
| **偷吃循环** | #6 谁偷吃了、#7、#9、#60 芒果偷吃鬼 | 设陷阱→偷吃被抓→死不承认→加码设陷阱→再被抓 |
| **成瘾循环** | #2 辣椒成瘾、#3 柠檬、#4 冰块、#5 跳舞、#115 | 极端行为→被制止→假意服从→暗中继续→揭示深层动机 |
| **诅咒/能力递进** | #41 点石成金、#122 触物成冰、#106 冰封 | 获得能力→小测试→中测试→大测试→失控/进化 |
| **三段模仿** | #83 撒泼打滚、#123 Buy IT | 人物A做→人物B学做→人物C学做→最后一个崩坏 |

**公式核心**：观众的期待感被**循环建立**，关键在每轮的**微变量升级**（更大的道具、更夸张的后果）和最后一轮的**规则打破**。

**公式核心**：前2/3时间给观众建立一个**错误认知**，最后用一个**关键证据/动作**瞬间推翻，完成情绪的急速翻转。

### 公式③：隐忍→爆发式（出现约 10%）

> **压力源施加→被迫忍耐→忍耐升级→到达极限→壮烈释放**

| 代表视频 | 结构 |
|----------|------|
| #16 忍痛轮椅逃亡-汤版 | 脚被烫→不能叫→轮椅逃亡→到雪山→声波爆发 |
| #17 忍痛轮椅逃亡-蛋版 | 脚被烫→不能叫→轮椅狂飙→冰山→仰天爆发 |
| #38 丛林砸手大逃亡 | 手指被砸→不能叫（有老虎）→抱着疯跑→安全后惨叫 |

**公式核心**：给主角设置一个**"不能释放"的约束**，让观众替主角憋得难受，释放那一刻制造**爆棚的代入感**。压力越荒诞，释放越壮烈。

## 三、跨公式的通用规律

### 3.1 开头铁律：1~3秒必须制造异常

| 开头类型 | 占比 | 示例 |
|----------|------|------|
| **异常行为** | ~45% | "疯狂吃辣椒"、"咬轮胎"、"涂脱毛膏" |
| **异常物体** | ~25% | "三扇小门"、"金色锡纸盲盒"、"紫色传送门" |
| **冲突爆发** | ~20% | "怒吼质问"、"争夺包包"、"推倒蛋糕" |
| **结果前置** | ~10% | "胸口插箭瘫坐"、"冰封病房醒来" |

> 没有任何一条骨架是用"日常平静画面"开头的。**第一个节拍就是 Hook**。

### 3.2 结尾铁律：最后一拍必须是情绪极值

| 结尾类型 | 占比 | 效果 |
|----------|------|------|
| **崩溃/绝望** | ~35% | jinu三连败绝望、mira当众崩溃 |
| **反转揭示** | ~25% | "其实是Mira伪装"、"原来是假蜘蛛" |
| **爆发释放** | ~15% | 声波碎冰山、蘑菇云升天 |
| **温情治愈** | ~15% | 重新微笑、双向救赎 |
| **荒诞定格** | ~10% | 两嘴对视、穿粉裙和解 |

### 3.3 节奏公式：2-4-8 加速曲线

分析所有节拍的时长分布，发现一条清晰的规律：

```
开头钩子:    1~3秒（极短，快速抓取注意力）
中段铺垫:    3~6秒（建立情境和规则）
高潮转折:    4~8秒（情绪释放，拍摄最精细）
结尾定格:    2~5秒（收束，但不拖泥带水）
```

- **总时长 < 20s** 的视频通常只有 4~5 个节拍，是"一个笑点"的体量（如 #19 巨型榴莲 8s、#31 婴儿认妈 7s）
- **总时长 30~50s** 的视频是主力区间，通常 7~10 个节拍，"建立→递进→反转"完整三段
- **总时长 > 55s** 的视频通常是"成瘾系列"或"善恶对照"等需要情感铺垫的类型

### 3.4 情绪曲线：三种主导模式

```
模式A: 搞笑型 📈📈📈💥
  紧张→更紧张→超级紧张→笑到崩溃
  （适用于竞赛循环、偷吃循环、隐忍爆发）

模式B: 反转型 📈📉💥
  错信→深信→一击粉碎→恍然大悟
  （适用于伪装揭穿、身份反转、骗局揭穿）

模式C: 治愈型 📉📉📈🌟
  被欺负→更惨→被善良的人发现→修复新生
  （适用于善恶对照、人偶救赎、成瘾救赎）
```

## 四、万能骨架模板

综合以上分析，可提炼出一个**可填充的骨架生成模板**：

```
[Hook异常画面](1-3s)
→ [建立规则/情境](3-5s)
→ [第一轮测试/尝试](4-6s)
→ [第二轮升级/更大赌注](4-8s)
→ [（可选）第三轮极致升级](4-8s)
→ [转折点/反转/爆发](4-8s)
→ [情绪收束/定格](2-5s)
```

### 变量填充表

| 变量 | 可选项 |
|------|--------|
| Hook类型 | 异常行为 / 异常物体 / 冲突爆发 / 结果前置 |
| 递进维度 | 尺寸递增 / 代价递增 / 人数递增 / 荒诞度递增 |
| 反转类型 | 身份揭穿 / 规则打破 / 视角切换 / 真相揭示 |
| 结尾情绪 | 崩溃 / 治愈 / 爆发 / 荒诞定格 / 温情和解 |
| 角色分工 | 发起者+受害者 / 竞赛双方 / 善方+恶方 / 误会双方 |

## 六、核心结论

> [!IMPORTANT]
> **这120条骨架的最底层规律是一个词：递进。**
> 
> 不管是循环、反转还是灾难链，结构上都遵循"每个节拍比上一个更极端"的原则。观众的多巴胺来自于"我猜到了下一步会更夸张，但没猜到*怎么*夸张"。

1. **没有平叙**：骨架里找不到任何"平铺直叙"的段落，每一拍都有动作变化
2. **没有过渡**：节拍之间是硬切，不存在"走在路上思考人生"这种空拍
3. **没有独白**：骨架只记录"发生了什么"（动作），不记录"说了什么"（台词）
4. **极度压缩**：平均每个节拍只有 3~5 秒，信息密度极高
5. **结尾必反**：90%以上的骨架结尾与开头的情绪方向相反（开头搞笑→结尾崩溃，开头悲惨→结尾治愈）
