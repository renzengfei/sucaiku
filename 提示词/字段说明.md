# 素材库字段说明

| 开头标签     | `hook_tags`       | 文本  | 开头抓住用户的方式，逗号分隔。可选：异常行为、异常物品、疑问、冲突、惊险。异常行为=打破常规的动作（吃辣椒、啃冰块）；异常物品=不合常理的物品（巨型榴莲、迷你门）                            | 异常行为, 冲突                                                                                     |

| 机制链条     | `mechanism`       | 文本  | 剥离所有表面元素（角色、场景、道具）后，视频"凭什么好看"的最底层运转逻辑。判断方法：把角色/场景/道具全换掉，什么不能换？不能换的就是机制链条                                     | 痛苦触发→不能叫的理由→忍耐累积→爆发                                                                          |

## 子表 `scenes`（一个视频可以有多个场景）

| 字段   | 数据库列名  | 说明            | 示例          |
| ---- | ------ | ------------- | ----------- |
| 场景名称 | `name`     | 场景的简称，需体现场景特点           | 杂乱厨房                  |

## 子表 `props`（一个视频可以有多个道具）

| 字段   | 数据库列名      | 说明          | 示例           |
| ---- | ---------- | ----------- | ------------ |

## 子表 `characters`（一个视频可以有多个角色）

| 字段   | 数据库列名       | 说明      | 示例      |
| ---- | ----------- | ------- | ------- |
| 角色名称 | `name`      | 角色名或类型  | 紫发女子    |

## 标签定义

### 开头标签（hook_tags）

> ⚠️ **以下标签为临时版本**，基于批量分析初步梳理。后续需逐个审视视频时根据实际情况修改，标签分类和定义可能会调整、合并或新增。
#### 动作层 — "谁在做什么不对劲的事"

| 标签   | 定义                          | 底层逻辑                  | 判断标准                   | 通道           | 一句话解释                         |
| ---- | --------------------------- | --------------------- | ---------------------- | ------------ | ----------------------------- |
| 异常行为 | 角色做了正常人不会做的事                | 违背常识→大脑启动预测           | 吃辣椒、啃冰块、猛推蛋糕           | 🧠 多巴胺（想知道/想得到） | 打破社会规范的行为，触发大脑的"异常检测系统"       |
| 异常物品 | 不该出现在该场景的物品                 | 视觉错位→"怎么会在这"          | 巨型轮胎在草地、银锅扣头骑车         | 🧠 多巴胺（想知道/想得到） | 与环境格格不入的元素触发大脑的"错误检测"，引发好奇或发笑 |
| 异常场景 | 呈现超现实、非日常或极具奇观的世界观设定        | 视觉奇观震慑→"这是哪里"         | 蒸汽朋克火车、末世避难所、火山口       | 🧠 多巴胺（想知道/想得到） | 用奇观震慑观众，《阿凡达》《星球大战》的核心卖点      |
| 身体异常 | 角色身体局部或整体处于非人类常态            | 共情本能→"他怎么了"           | 全身冰晶覆盖、下半身被漩涡吸入、腹部出现黑洞 | 🧠 多巴胺（想知道/想得到） | 夸张扭曲的人体形态，同时引发惊奇、不适和着迷        |
| 突发意外 | 角色遭遇意料之外的物理倒霉事件             | 幸灾乐祸/惊吓反转→"好惨但好好笑"    | 摔了一跤满脸酱、袋子破裂水果砸头、突然噎住  | ⚡ 皮质醇（威胁警报）  | 物理倒霉事件——《猫和老鼠》的全部精髓           |
| 滑稽疼痛 | 角色遭受突如其来的物理打击并伴随极其夸张的痛觉反应   | 痛觉通感+安全笑点→"看着都疼但好好笑"  | 煎蛋精准烫光脚、被螃蟹死死夹住手       | 🎁 内啡肽（奖赏预期） | 看别人倒霉时产生的快感，人类最古老的娱乐方式        |
| 冲突对峙 | 两个角色有明确的利益对立                | 社会脑激活→"谁会赢"           | 抢叉子、威胁扔LEGO            | 🧠 多巴胺（想知道/想得到） | 编剧圣经的核心：没有冲突就没有故事             |
| 亲密互动 | 角色间展现亲吻、求婚、撒娇等极其甜蜜的行为       | 建立美好预期→等待被打破的反差       | 甜蜜对视、亲吻脸颊/手指           | 🤝 催产素（社会共情） | 用极致甜蜜建立"一切都很美好"的假象，为后续反转蓄力    |
| 急迫   | 角色带着极度焦急、仓皇、不顾一切的肢体动作冲向某个目标 | 皮质醇传染→"她怎么了/后面有什么在追她" | 满身灰尘冲进大堂猛灌水、跌跌撞撞推门乞求   | ⚡ 皮质醇（威胁警报）  | 炸弹倒计时、末班车要开走——时间压迫感锁死注意力      |
| 竞赛与挑战   | 角色进入有明确胜负规则的比赛，或单人极限硬核挑战         | 赌博心理→自动站队下注→"能否成功/谁会赢"     | 吹蜡烛对决、柔弱钻铁窗越狱、吃极度辣的东西挑战     | 🧠 多巴胺（想知道/想得到） | 激发观众"等待开奖/下注"的赌博心理，完播率之王           |

#### 信息层 — "观众的大脑在做什么"

| 标签     | 定义                            | 底层逻辑                           | 判断标准                                           | 通道              | 一句话解释                        |
| ------ | ----------------------------- | ------------------------------ | ---------------------------------------------- | --------------- | ---------------------------- |
| 提出疑问   | 角色主动发问或画面暗示悬念                 | 信息缺口→"答案是什么"                   | "谁吃了我的火龙果"                                     | 🧠 多巴胺（想知道/想得到） | 信息缺口理论：人一旦知道问题存在，就无法忍受不知道答案  |
| 经典桥段   | 模仿绑架/凶杀/绝症等大片感的高压场景           | 经典套路触发危险认知→破功反转                | 黑车刹车跳下壮汉、如临大敌只为剪刘海                             | ⚡ 皮质醇（威胁警报）     | 用观众熟悉的高压经典场景当诱饵，再突然换成搞笑内容    |
| 视觉目标绑架 | 给观众下发一个1秒内可见结果的物理微目标          | 强迫症闭环本能→"不看到结果决不划走"            | 抛糖果入嘴、水倒满杯沿、极限拉伸橡皮泥                            | 🎁 内啡肽（奖赏预期）    | 未完成的任务会持续占据大脑工作记忆，直到看到结果     |
| 贫穷与美好  | 极度落魄肮脏的贫穷元素，与极其快乐或奢华美好的事物强行同框 | 阶层画风撕裂引发共情与震撼→"心酸治愈" 或 "阶级冒犯感" | 乞丐装扮送彩虹糖、穷人开心滑滑梯、脏污工装急迫冲进五星级大堂                 | 🤝 催产素（社会共情）    | 独创标签：用同一画面内的阶层视觉撕裂，同时触发悲悯和震撼 |
| 致命傲慢   | 角色极度自信地宣布必胜/必成功的嚣张宣言          | 打脸预期→"等着看你翻车"                  | "Big one's mine, you're going to lose"、"我肯定能赢" | 🎁 内啡肽（奖赏预期）    | 致命傲慢必遭天罚，2500年来最强的"完播率锁定器"   |
| 生命威胁   | 角色面临明显的生命危险或暗中物理入侵            | 皮质醇飙升/悬念恐惧→"快逃"或"完了"           | 丧尸包围、暴风雪侵袭、异物飞入口中                              | ⚡ 皮质醇（威胁警报）     | "炸弹在桌子底下"——观众知道危险存在但角色不知道    |
| 暗中算计   | 角色背地里冷酷地谋划伪造证据、栽赃或制造陷阱         | 信息不对称/上帝视角→"完了她要害人"           | 对镜自语策划阴谋、背地伪造伤口特效血迹、下药                      | ⚡ 皮质醇（威胁警报）     | 影视界顶级的悬念：观众知道反派的腹黑计划，但主角还蒙在鼓里 |
| 猎奇恶心   | 呈现极度肮脏、黏腻或带有污垢暗示的视觉元素         | 猎奇心态/防御性排斥→"太恶心了但又想看"      | 脏脚底摩擦食物、掏耳屎吃、布满黑色泥垢的脏手突然伸出                 | ⚡ 皮质醇（威胁警报）     | 利用极度反胃的画面，强制唤醒人类对病菌和脏东西的进化学防御本能 |


### 视频标签（video_tags）

| 标签 | 定义          | 判断标准             |
| -- | ----------- | ---------------- |
| 荒诞 | 超出现实逻辑的夸张设定 | 现实中不可能发生          |
| 搞笑 | 引发笑点        | 观众会笑出来            |
| 恐怖 | 制造恐惧感       | 观众感到害怕或不安         |
| 治愈 | 温暖正面的情感     | 观众看完心里暖暖的         |
| 感动 | 触动情感共鸣      | 观众看完想哭或被打动        |
| 竞赛 | 角色间有胜负对抗    | 有明确的赢/输结果或比拼过程    |

### 改编标签（adapt_tags）

| 标签         | 定义                                             | 示例                           |
| ---------- | ---------------------------------------------- | ---------------------------- |
| 换皮型改编      | 保留内核，替换表层视觉/行业/受众                              | 辣椒→柠檬，煎蛋→火龙烤脚                |
| 核心机制迁移型改编  | 提取最核心的爆点机制，应用于全新场景                             | "异常反转"从食物成瘾迁移到芭蕾舞特训          |
| 连续兑现型改编    | 将原视频的单次高潮，拆解并变为多次密集高潮                          | 一扇门→三扇门连续开                   |
| 递进升级型改编    | 在原视频逻辑基础上，加大筹码、风险或奖励                           | 猫砂惩罚→炸弹爆炸，追加鬼魂结尾             |
| 规则建立-打破型改编 | 前段复刻原视频建立预期，后段彻底打破制造极致反差                       | 前两扇门建立规则，第三扇打破               |
| 错误学习型改编    | 反向操作，展示"如果不按原视频做会怎样"的踩坑实录                      | 拔箭疼→插回去更疼→不知道自己错了            |
| 二次反转型改编    | 在原视频的反转点之上，再增加一层意料之外的逻辑反转                      | 以为安全了→更大危险                   |
| 场景嫁接型改编    | 将原视频的逻辑嫁接到毫不相干、甚至冲突的场景中                        | 厨房忍痛→火山洞穴忍痛，场景强化紧张感          |
| 关系重写型改编    | 改变原视频中人物/事物的权力结构或依附关系                          | 陌生人互整→情侣互整，增加感情张力            |
| 延伸式改编      | 顺着原视频剧情的结果继续向后延伸叙事，在延伸出的极端情境中让核心行为再次重现，证明其不可消灭 | 偷吃者被炸死→上坟→祭品汉堡消失（鬼魂仍在偷吃）     |
| 败者重生改编     | 开局继承母版中的失败经历，原来的失败者吸取教训变聪明，用全新手段反败为胜           | 偷吃被抓（母版）→学会用保鲜膜/掏空/拼图等诡计完美脱罪 |
