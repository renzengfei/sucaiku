在选题裂变/创意构思过程中，你可以从爆款视频素材库中查询参考素材。

素材库是一个 SQLite 数据库，存储了大量已拆解的爆款视频，包含场景、道具、角色、结构等维度。

---

## 数据库位置

```
/Users/renzengfei/短视频/素材库/database.db
```

## 查询方式

使用 `sqlite3` 命令行查询，以下是常用查询模板：

### 1. 浏览所有场景

当你需要场景灵感时：

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT DISTINCT s.name, s.description, v.name as video_name FROM scenes s JOIN videos v ON s.video_id=v.id ORDER BY s.name"
```

### 2. 浏览所有道具

当你需要道具灵感时：

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT DISTINCT p.name, p.function, v.name as video_name FROM props p JOIN videos v ON p.video_id=v.id ORDER BY p.name"
```

### 3. 浏览所有角色人设

当你需要人设参考时：

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT DISTINCT c.name, c.persona, c.abilities, c.states, v.name as video_name FROM characters c JOIN videos v ON c.video_id=v.id ORDER BY c.name"
```

### 4. 浏览所有结构

当你需要叙事结构参考时：

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT name, structure FROM videos WHERE structure != '' ORDER BY date DESC"
```

### 5. 搜索包含特定关键词的素材

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT v.name, s.name, s.description FROM scenes s JOIN videos v ON s.video_id=v.id WHERE s.name LIKE '%关键词%' OR s.description LIKE '%关键词%'"
```

### 6. 查看某个视频的完整信息

当你想深入了解某个视频是如何运用某个元素的：

```bash
sqlite3 /Users/renzengfei/短视频/素材库/database.db -header -column \
  "SELECT * FROM videos WHERE name='视频名称'"

sqlite3 /Users/renzengfei/短视频/素材库/database.db -header -column \
  "SELECT * FROM scenes WHERE video_id=(SELECT id FROM videos WHERE name='视频名称')"

sqlite3 /Users/renzengfei/短视频/素材库/database.db -header -column \
  "SELECT * FROM props WHERE video_id=(SELECT id FROM videos WHERE name='视频名称')"

sqlite3 /Users/renzengfei/短视频/素材库/database.db -header -column \
  "SELECT * FROM characters WHERE video_id=(SELECT id FROM videos WHERE name='视频名称')"
```

### 7. 查看原始脚本

如果需要参考原视频的完整脚本：

```bash
# 先查脚本路径
sqlite3 /Users/renzengfei/短视频/素材库/database.db \
  "SELECT script_path FROM videos WHERE name='视频名称'"

# 然后读取脚本文件（路径相对于素材库目录）
cat /Users/renzengfei/短视频/素材库/对应的script_path
```

## 使用策略

1. **先扫描再深入**：先用查询 1-4 快速浏览某个维度的所有素材，找到感兴趣的关键词
2. **按需深入**：发现有灵感的元素后，用查询 6 查看这个视频的完整信息，理解元素在原视频中的上下文
3. **交叉参考**：如果发现某个场景很有趣，可以继续查看使用这个场景的视频都用了什么道具和角色，寻找搭配规律

> ⚠️ 如果查询结果为空，说明数据库中还没有相关素材。这不影响创作，素材库是参考而非限制。
